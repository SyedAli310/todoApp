# Todo App
A todo application with filters to keep you organized.

## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedAli310/todoApp.git
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  python manage.py runserver
```
